# jokenp-Java

Jokenpô

Projeto interativo de Pedra, Papel e Tesoura desenvolvido para praticar lógica de programação e manipulação de elementos com JavaScript.

Funcionalidades

- Escolha entre pedra, papel ou tesoura
- Resultado automático da partida
- Atualização dinâmica da interface

Tecnologias utilizadas:

- HTML5
- CSS3
- JavaScript

Objetivo do projeto

Praticar lógica condicional, manipulação de DOM e interatividade com JavaScript.


Aprendizados

Neste projeto, desenvolvi habilidades com:

- Estruturas condicionais
- Eventos de clique
- Atualização dinâmica de conteúdo
- Organização de lógica JavaScript
